# xsizero

Joc Tic-Tac-Toe creat in Flutter. În xsizero utilizatorul poate juca împotriva unui alt utilizator sau împotriva calculatorului. Acesta are două dificultăți: ușor (mută aleatoriu) și dificil (Minimax).
